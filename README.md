# Nails Galore and More

Nails Galore and More is a booking website that can be used to book an appointment or service at the Nail Galore and More Shop. It also displays the different nail services and the prices for each service.

## Usage
Use the GitHub link [NailGaloreandMore](hgit@github.com:joipeterson1/phase-1-project.git) to access the repository through the terminal

```bash
git clone git@github.com:joipeterson1/phase-1-project.git

. code

```
Open the index.html file in the browser
```
open index.html
```
